SÉCURITÉ WINDOWS

Désactivez Windows Defender complètement pour éviter tout blocage :

  • Menu Démarrer → "Sécurité Windows"
  • Protection contre les virus → Gérer les paramètres
  • Désactivez : Protection en temps réel
  • Désactivez : Cloud, Altérations & Échantillons
02

COMPOSANTS SYSTÈME

Installez les runtimes obligatoires pour éviter les erreurs .DLL :

Visual C++ Redist DirectX Runtime AIO Runtimes
03

CONFIGURATION NOYAU

UAC : Cherchez "UAC" et mettez le curseur au minimum.

ISOLATION : Cherchez "Isolation du noyau" → Activez l'Intégrité de la mémoire.

Cliquez ci-dessous pour copier le chemin du dossier à nettoyer :

C:\Windows\System32\drivers\etc

Supprimez le fichier hosts à l'intérieur.

REDÉMARRAGE REQUIS
04

PROCÉDURE VALORANT

1 Lancez Valorant → Vidéo → Fenêtré sans bordure.
2 Désinstallez Vanguard via Revo Uninstaller (Mode Avancé).
3 Ouvrez le Loader → Clé de licence → Redémarrez.
4 Ouvrez le Riot Client → Laissez Vanguard se réinstaller.
5 Redémarrez le PC une dernière fois.
6 Lancez le Loader → Attendez "Waiting for Val" → Lancez le jeu.
